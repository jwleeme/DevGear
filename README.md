## 📌 주요 기능

- 네비게이션 바를 통해 카테고리, 로그인, 검색 제공
- 로그인은 JWT를 통해 암호화
- 카테고리는 2계층으로 구성
- 상품 리스트는 페이징 및 정렬 가능
- 상품 장바구니 추가 및 주문 가능
- 관리자를 통해 상품, 카테고리, 유저, 주문 데이터에 대해 CRUD
- Nginx를 통해 프록시 설정
- HTTPS (SSL) 설정
